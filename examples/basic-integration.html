Basic Integration Example

Current Version: 1.0.0

This is a simple example showing how to integrate the Auto-Update System into your static website. The system will automatically check for updates every 30 seconds and notify users when a new version is available.

Features Demonstrated:

  • Automatic update detection
  • User-friendly notifications
  • Manual update checks
  • Version display
How to test:
1. Open version-manifest.json
2. Change the version number (e.g., from 1.0.0 to 1.0.1)
3. Click "Check for Updates" or wait 30 seconds
4. You'll see an update notification appear
