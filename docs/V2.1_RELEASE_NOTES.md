# Version 2.1.0 Release Notes

## What's New

Version 2.1.0 adds framework integrations, TypeScript support, build tool plugins, and progressive rollout capabilities.

### TypeScript Definitions ✅

Complete TypeScript definitions with full type safety:

```typescript
import type { AutoUpdate } from 'auto-update';

const config: AutoUpdate.Config = {
  manifestUrl: '/version-manifest.json',
  checkInterval: 30000,
  rolloutPercentage: 0.5,
  onUpdateAvailable: (newVersion: string, oldVersion: string) => {
    console.log(`Update: ${oldVersion} → ${newVersion}`);
  }
};

AutoUpdate.init(config);
```

Works with TypeScript, VS Code IntelliSense, and other editors.

### Framework Integrations ✅

Ready-to-use integrations for popular frameworks:

**React**
```jsx
import { useAutoUpdate } from './hooks/useAutoUpdate';

function App() {
  const { updateAvailable, applyUpdate } = useAutoUpdate({
    manifestUrl: '/version-manifest.json'
  });

  return (
    <div>
      {updateAvailable && (
        <button onClick={applyUpdate}>Update Now</button>
      )}
    </div>
  );
}
```

**Vue**
```vue
<script setup>
import { useAutoUpdate } from './composables/useAutoUpdate';

const { updateAvailable, applyUpdate } = useAutoUpdate({
  manifestUrl: '/version-manifest.json'
});
</script>

<template>
  <button v-if="updateAvailable" @click="applyUpdate">
    Update Now
  </button>
</template>
```

**Angular**
```typescript
@Component({
  selector: 'app-root',
  template: `
    <button *ngIf="updateAvailable$ | async" (click)="applyUpdate()">
      Update Now
    </button>
  `
})
export class AppComponent {
  updateAvailable$ = this.autoUpdate.getUpdateAvailable();
  
  constructor(private autoUpdate: AutoUpdateService) {}
  
  applyUpdate(): void {
    this.autoUpdate.applyUpdate();
  }
}
```

### Build Tool Plugins ✅

Automatic manifest generation during build:

**Webpack**
```javascript
const AutoUpdateWebpackPlugin = require('./plugins/webpack-plugin');

module.exports = {
  plugins: [
    new AutoUpdateWebpackPlugin({
      version: '1.0.0',
      files: ['**/*.js', '**/*.css']
    })
  ]
};
```

**Vite**
```javascript
import autoUpdate from './plugins/vite-plugin';

export default defineConfig({
  plugins: [
    autoUpdate({
      version: '1.0.0',
      files: ['**/*.js', '**/*.css']
    })
  ]
});
```

No more manual version bumping - the plugins handle it automatically.

### Progressive Rollout ✅

Update only a percentage of users:

```javascript
AutoUpdate.init({
  manifestUrl: '/version-manifest.json',
  rolloutPercentage: 0.1  // 10% of users
});
```

Perfect for:
- Testing new versions with a subset of users
- Gradual rollouts to minimize risk
- A/B testing different versions

Each user gets a stable ID, so they consistently either get or don't get the update.

1. **TypeScript**: Copy `auto-update.d.ts` to your project
2. **Frameworks**: Copy the example for your framework
3. **Build Plugins**: Copy the plugin for your build tool
4. **Progressive Rollout**: Add `rolloutPercentage` to your config

## Examples

### Progressive Rollout Example

Start with 10% of users:

```javascript
AutoUpdate.init({
  manifestUrl: '/version-manifest.json',
  rolloutPercentage: 0.1
});
```

After testing, increase to 50%:

```javascript
rolloutPercentage: 0.5
```

Finally, full rollout:

```javascript
rolloutPercentage: 1.0
```

### React Example

```jsx
import { useAutoUpdate } from './hooks/useAutoUpdate';

function App() {
  const {
    updateAvailable,
    currentVersion,
    newVersion,
    applyUpdate,
    checkNow
  } = useAutoUpdate({
    manifestUrl: '/version-manifest.json',
    rolloutPercentage: 0.5  // 50% rollout
  });

  return (
    <div>
      <header>
        <h1>My App</h1>
        <span>Version: {currentVersion}</span>
        <button onClick={checkNow}>Check Updates</button>
      </header>

      {updateAvailable && (
        <div className="update-banner">
          <p>Version {newVersion} available!</p>
          <button onClick={applyUpdate}>Update Now</button>
        </div>
      )}
    </div>
  );
}
```

### Webpack Plugin Example

```javascript
const AutoUpdateWebpackPlugin = require('./plugins/webpack-plugin');
const package = require('./package.json');

module.exports = {
  plugins: [
    new AutoUpdateWebpackPlugin({
      version: package.version,
      files: ['**/*.js', '**/*.css'],
      exclude: ['**/*.map'],
      description: `Build ${package.version}`
    })
  ]
};
```
